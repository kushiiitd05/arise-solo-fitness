**Edit 1 — Fix DISMISS_NOTIFICATION (line 314):**

Replace:
```typescript
case "DISMISS_NOTIFICATION":
  return { ...state, notifications: state.notifications.map(n => n.id === action.payload ? { ...n, read: true } : n) };
```

With:
```typescript
case "DISMISS_NOTIFICATION":
  return { ...state, notifications: state.notifications.filter(n => n.id !== action.payload) };
```

Why: AnimatePresence tracks items by key={n.id}. When an item stays in the array with read: true, React never unmounts it, so the exit animation never fires and the notification stays visible forever. Filtering removes the item, React unmounts it, AnimatePresence fires the exit animation.
